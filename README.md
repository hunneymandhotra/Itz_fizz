# ITZ FIZZ | Scroll-Driven Hero Section

This project is a high-performance, scroll-driven hero section animation built as part of the Frontend Developer assessment.

## 🚀 Experience the Motion
The project features a **smooth, scroll-bound animation** centered around a futuristic supercar, designed with a premium dark-mode aesthetic and fluid interactions.

### Features:
- **Scroll-Tied Animation**: Car movement (Y-pos), rotation, and scale are fully synchronized with the user's scroll progress using **GSAP ScrollTrigger**.
- **Initial Load Stagger**: Headline "WELCOME ITZ FIZZ" and key metrics reveal themselves with a staggered premium animation.
- **Micro-Interactions**: Custom glowing cursor and interactive hover states for statistics.
- **Tech Stack**: Next.js 15, Tailwind CSS v4, GSAP, and Lucide Icons.

## 🛠️ Local Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Visit `http://localhost:3000`.

## 📦 Deployment
The project is optimized for **Vercel**. Simply connect your GitHub repository to Vercel for instant deployment.
